Nama aplikasi: Quiz
Deskripsi: Aplikasi Quiz menampilkan soal pilihan ganda dan pembahasan kepada user. Aplikasi ini bersifat pilihan ganda, 

dimana user ditentukan untuk memilih jawaban dari 4 pilihan (A, B, C, D) yang berupa pertanyaan umum. 

Hasil dari quiz ini, user dapat melihat jawaban benar dan salah sesuai dengan yang dijawab oleh user, sehingga user mengetahui letak kesalahan dari soal.
